Change Datacenter 

If you are having high ping when using playit make sure that your playit program connects to the closest Datacenter to you you can check this on the playit website

You can change your location if you have IPv6 https://test-ipv6.com/
To do this put one of the control addresses at the top of your playit.toml file

On Linux
you can find this file /etc/playit/playit.toml,
On Windows,
it is located in the same place as your playit.exe

The Playit agent should route you in the best way possible.

Have a bad ping? No Ipv6

READ FIRST
It can be really difficult to update routing.
Routing is something your ISP decides, not us.
There are a few things we can try to influence your ISP to route you better,
but for many, it won't be possible until we buy more IP space.

Send an email to support@playit.gg with the following data:
1) Where you're located (country/city/state)
2) The text from http://ping.playit.gg/
3) Which data center do you think would be best, visit https://playit.gg/about and see the map
4) One of your playit.gg tunnel addresses
