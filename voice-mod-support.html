How to use Simple Voice chat with Playit.gg

Part One

Go to playit.gg.
Create a new custom UDP tunnel with the add tunnel button:

It should look something like this:
adjective-noun.auto.playit.gg:59931

Edit the tunnel and change the port number, the number after 127.0.0.1,
to the auto-generated port number at the end of your new tunnel, in this case, ‚Äú59931‚Äù

This UDP tunnel is only for voice chat. When you join a server with your local IP address or TCP Playit tunnel, Simple Voice Chat will tell your client mod to connect the voice chat over a separate UDP tunnel that you enter in voice_host in Part Two you are going to be using two separate tunnels at once.

Part Two

  1. Installing Voice Chat Mode
    Install Simple Voice Chat on your server.
    You can download it from the curse forge website: Simple voice chat.
    If you are using the spigot version put the file in your plugins folder,
    if using Forge/Fabric put the file in your mods folder.
    You will also need the¬†ProtocolLib plugin to use the spigot version.
  2. Restart you Minecraft server
  3. Find and open voicechat-server.properties.
    It will be in the voice chat folder in your plugins/mods folder
  4. Change the port value to the port number at the end of your new UDP Playit tunnel.
    In this case port=59931.
  5. Change the ‚Äò‚Äòvoice_host‚Äô‚Äô value to your UDP tunnel address, in this case, ‚Äúvoice_host=adjective-noun.auto.playit.gg‚Äù
  6. Restart your Minecraft server
  7. Install the clientside Simple Voice Chat mod in your Minecraft mods folder and join your server.
  8. Press ‚ÄúV‚Äù for settings and to check the connection is working.

Done!
